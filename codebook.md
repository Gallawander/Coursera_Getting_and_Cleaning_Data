# Code Book

This codebook provides detailed information regarding the dataset in the `tidy.txt` file. \
It summarizes and describes all the features contained in this dataset.

## Identifiers of the Dataset

* `Subject` - The ID of the test subject
* `Activity` - The ID of an activity performed by the subject
* `ActName` - The type of an activity corresponding to the activity ID

## Activity Labels

* `WALKING` - corresponds to `Activity` = `1`
* `WALKING_UPSTAIRS` - corresponds to `Activity` = `2`
* `WALKING_DOWNSTAIRS` - corresponds to `Activity` = `3`
* `SITTING` - corresponds to `Activity` = `4`
* `STANDING` - corresponds to `Activity` = `5`
* `LAYING` - corresponds to `Activity` = `6`
